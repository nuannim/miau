This is example for using Partii with NodeJS and Web service.
Quick start for gRPC node-js can go to https://grpc.io/docs/languages/node/quickstart/

In this example show you how to record audio on web browser and stream to Partii service :
Before use this example you need to fix your apikey in index.js first.
Then you can start web service with this command "node index.js".
After that you can access the website via http://localhost:3000 and start|stop record audio.
